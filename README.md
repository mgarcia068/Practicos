# Repositorio de Ejercicios Prácticos en Java

Este repositorio contiene **9 ejercicios prácticos realizados en Java**, diseñados para reforzar conceptos fundamentales de programación y estructuras de datos, así como la implementación de pequeños sistemas. Cada ejercicio aborda un tema específico, desde fundamentos de programación orientada a objetos hasta estructuras de datos avanzadas.

---

## Contenido del repositorio

1. **Sistema Gestor de Tareas**  
   Desarrollo de un sistema simple de gestión de tareas, utilizando estructuras de datos y conceptos de programación orientada a objetos.

2. **Sistema de Gestión de Pizzería**  
   Implementación de un sistema que permite gestionar pedidos, clientes y productos de una pizzería, aplicando clases, listas y lógica de negocio.

3. **Clases**  
   Implementación de ejercicios básicos utilizando clases y objetos en Java, incluyendo atributos, métodos y constructores.

4. **Recursividad**  
   Ejercicios que aplican técnicas recursivas para resolver problemas clásicos de programación.

5. **Pila y Cola con Arreglos**  
   Implementación de estructuras de datos lineales utilizando arreglos: pilas (stack) y colas (queue).

6. **Lista Enlazada Simple**  
   Ejercicios que demuestran el manejo de listas enlazadas simples, con inserción, eliminación y recorrido de nodos.

7. **Árbol AVL**  
   Implementación de árboles binarios balanceados AVL, con operaciones de inserción, eliminación y rotaciones automáticas.

8. **Árbol Rojinegro**  
   Ejercicios sobre árboles binarios de búsqueda balanceados (Red-Black Tree), incluyendo inserción y balanceo.

9. **Montículo Binario (Heap)**  
   Implementación de montículos binarios para manejo de prioridades, con operaciones de inserción y extracción.
---

## Tecnologías utilizadas

- Java 8 o superior
- Estructuras de datos: arreglos, listas enlazadas, pilas, colas, árboles
- Programación orientada a objetos
- Conceptos de recursividad y algoritmos básicos

Cada carpeta contiene el código fuente (`.java`) correspondiente al ejercicio práctico.
